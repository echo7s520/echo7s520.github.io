数据

插入

INSERT INTO '表名' VALUES（）

更新

UPDATE '表名' SET

删除

DELETE FROM '表名' WHERE

查询

去重、别名

SELECT [ DISTINCT ] * | {column1,column2..} FROM '表名'

column 指定列名。DISTINCT 去重

where

SELECT column1,column2...FROM table WHERE ...

运算符 符号 功能
比较运算符 > < <= >= = <> != 大于，小于，不等于…
BETWEEN…AND… 在某个区间（闭区间）
IN（集合） 显示在 in 列表中的值。in（100，200）
LIKE，NOT LIKE 模糊查询，像…
IS NULL 是否为空
逻辑运算符 AND 与运算
OR 或运算
NOT 非运算

order by

SELECT column1... FROM ... ORDER BY cloumn asc | desc

ORDER BY 指定排序的列，排序的可以为表中的列名，也可以为查询后的列名。

ASC —> 升序（默认），DESC —> 降序 ，ORDER BY 应位于结尾处。

count

SELECT COUNT(*) | COUNT( 列名 ) FROM table WHERE ...

count（*）和 count（列）的区别？

count（*）返回满足条件的记录的行数

count（列）返回满足条件的某列个数，但是会排除为 null 的情况

sum

SELECT SUM(列名)，{SUM(列名)} FROM table WHERE ...

avg

SELECT AVG(列名)，{AVG（列名）} FROM table WHERE ...

max、min

``SELECT MAX(列名) FROM table WHERE …

group by、having

使用 group by 语句对列进行分组

SELECT column1,column2.. FROM table GROUP BY column

使用 where 对分组前进行过滤

使用 having 语句对分组 后 结果进行过滤

SELECT column1... FROM table GROUP BY column1 HAVING

字符

函数 功能
charset（str） 返回字串字符集（utf8）
concat（string2，[ … ] ) 连接字符串
insrt（str1，str2） 返回str2在 str1中出现的位置
ucase（string） 转换为大写
lcase（string） 转化为小写
left（string，length） 从 string 左边起取 length 个字符
length（string） string 长度（按字节）
replace（str，str1，str2） 在str 中使用 str2 替换 str1
strcmp（str1，str2） 逐字符比较两字串大小
substring（str，position [ ,length ]） 从str的 position 开始计算取length个字符
ltrim（str2）、rtrim（str2）、trim 去除前后端空格

数学

函数 功能
abs（num） 绝对值
bin（decimal_number） 十进制转二进制
ceiling（num） 向上取整
floor（num） 向下取整
conv（num，from_base，to_base） 进制转换
format（num，decimal_places） 保留小数位数
hex（num） 转十六进制
least（num1，num2，…） 求最小值
mod（num，denominator） 求余
rand（） 随机数 [ 0 - 1 ]

日期

函数 功能
current_date（） 当前日期
current_time（） 当前时间
current_timestamp 当前时间戳
date（datetime） 返回datetime的日期部分
date_add（date，v） 在 date 中加上日期或时间
date_sub（date，v） 在 date 中减去日期或时间
datediff（date1，date2） 两个日期差（结果为天）
timediff（date1，date2） 两个时间差（结果为时分秒）
now（） 当前时间

加密、系统函数

函数 功能
user（） 查询用户
database（） 数据库名称

password（str） 从原文密码str计算返回密码字符串

流程控制

函数 功能
if（expr1，expr2，expr3） 如果 expr1为true，则返回expr2，否则返回expr3
ifnull（expr1，expr2） 如果expr1 不为空，返回主键自己，否则返回expr2

总结—分组

如果 SELECT 语句同时包含 group by，having，limit，order by。那么它们的顺序是 group by ，having，order by，limit

多表查询

自连接

子连接就是在同一张表中的连接查询（把一张看作两张表）

子查询

子查询：子查询就是指嵌入在其它 sql 语句中的 select 语句，也叫嵌套查询

单行子查询：单行子查询指返回一行数据的子查询语句

多行子查询：多行子查询指返回多行数据的子查询

多列子查询：多列子查询指返回多列数据的子查询

合并查询

union all

union all：该操作符用于取得两个结果集的并集，使用该操作符不会取消重复行

union

union：该操作符和 union all 相似，但是 会自动取消掉结果集中重复行

外连接

左外连接

左外连接：如果左侧的表完全显示，就是左外连接

select ... from 左表 left join 右表 on 条件

右外连接

右外连接：如果右侧的表完全显示，就是右外连接

select ... from 左表 right join 右表 on 条件

约束 功能
